SQL SELECT Statement

คำสั่ง SELECT ใช้สำหรับเลือกข้อมูลจากฐานข้อมูล ข้อมูลที่เลือกจะถูกจัดเก็บในตารางผลลัพธ์ (Result-Set) ซึ่งเรียกว่าชุดเรคคอร์ด (recordset).

ไวยากรณ์:

                
                    SELECT column1, column2, ...
                    FROM table_name;

หรือเพื่อเลือกคอลัมน์ทั้งหมดจากตาราง:

                
                    SELECT *
                    FROM table_name;

ตัวอย่างการใช้งาน:

สมมติว่าเรามีตาราง Customers ที่มีคอลัมน์ CustomerID, CustomerName, ContactName, City:

                
                    SELECT CustomerName, City
                    FROM Customers;

คำสั่งข้างต้นจะเลือกเฉพาะชื่อลูกค้าและเมืองจากตาราง Customers.

SELECT DISTINCT:

ใช้สำหรับเลือกค่าที่แตกต่างกัน (unique values) เท่านั้น

                
                    SELECT DISTINCT Country
                    FROM Customers;
