SQL Introduction

SQL ย่อมาจาก Structured Query Language เป็นภาษามาตรฐานสำหรับจัดการข้อมูลในฐานข้อมูลเชิงสัมพันธ์ (Relational Database Management Systems - RDBMS).

SQL ใช้ทำอะไรได้บ้าง?

  • สอบถามข้อมูลจากฐานข้อมูล
  • แทรก (Insert) บันทึกข้อมูล
  • อัปเดต (Update) บันทึกข้อมูล
  • ลบ (Delete) บันทึกข้อมูล
  • สร้าง (Create) ฐานข้อมูลใหม่
  • สร้าง (Create) ตารางใหม่ในฐานข้อมูล
  • สร้าง (Create) มุมมอง (Views)
  • กำหนดสิทธิ์ (Permissions) ในฐานข้อมูล

RDBMS ยอดนิยมที่ใช้ SQL:

  • MySQL
  • Microsoft SQL Server
  • PostgreSQL
  • Oracle
  • IBM DB2
