POS Starter (HTML + JS) for students
-----------------------------------
วิธีใช้งาน:
1) วางโฟลเดอร์ pos_starter ลงใน C:\xampp\htdocs (บน Windows) หรือ /Applications/XAMPP/htdocs (macOS)
2) เปิด XAMPP Control Panel แล้ว Start Apache
3) เปิดเบราว์เซอร์ ไปที่: http://localhost/test/
4) ทดสอบเพิ่มสินค้า ใส่จำนวน กดบันทึก/พิมพ์ หรือ Export CSV

ไฟล์ในชุด:
- index.html : หน้าเว็บหลัก
- style.css  : รูปแบบ (CSS)
- app.js     : ฟังก์ชันระบบขาย (JavaScript)
- products.json : รายการสินค้า (สำรอง)
- README.txt : คำแนะนำการใช้งาน

หมายเหตุ:
- ระบบนี้เป็นตัวอย่างฝึกใช้งานเท่านั้น (client-side) ไม่มีฐานข้อมูล
- หากต้องการเก็บข้อมูลจริง ให้เชื่อมต่อกับ backend (PHP + MySQL) ต่อไป
