📖 วิธีตั้งค่า Google API
📖 คู่มือตั้งค่า Google API
1
สร้าง Google Cloud Project

ไปที่ console.cloud.google.com → สร้าง Project ใหม่ → เปิดใช้งาน Google Drive API และ Google Sheets API

2
สร้าง OAuth 2.0 Client ID

APIs & Services → Credentials → Create Credentials → OAuth Client ID → Web Application → ใส่ https://yourdomain.com ใน Authorized Origins

3
สร้าง Google Sheet

สร้าง Google Sheet ใหม่ → Copy ID จาก URL (docs.google.com/spreadsheets/d/SHEET_ID/edit) → ตั้งชื่อ Sheet: teachers, tasks, submissions

4
สร้างโฟลเดอร์ใน Google Drive

สร้างโฟลเดอร์สำหรับเก็บเอกสาร → Share โฟลเดอร์กับ Google Account ทุกครู → Copy Folder ID ใส่ในระบบ

5
Deploy บน GitHub Pages หรือ Netlify

อัปโหลดไฟล์ HTML ขึ้น GitHub Pages ฟรี → ใส่ URL ที่ได้ใน Authorized Origins ของ OAuth Client
