📋 ใบ ปพ. คืออะไร?

ใบ ปพ. คือ เอกสารประเมินผลตามหลักสูตรการศึกษาขั้นพื้นฐาน ที่บ่งบอกข้อมูลทางการศึกษาของนักเรียน เพื่อใช้เป็นหลักฐานในการจัดการเรื่องต่าง ๆ ที่เกี่ยวกับการศึกษา มีทั้งหมด 9 ประเภท แต่ละประเภทมีจุดประสงค์ที่แตกต่างกัน

📊 ตารางสรุป

ใบ ปพ. แต่ละประเภท ใช้ทำอะไร?

ปพ. ชื่อเอกสาร รายละเอียด ผู้ได้รับ แท็ก
1
ใบแสดงผลการเรียน
Transcript
 แสดงรายวิชา หน่วยกิต และเกรดทุกวิชา ใช้สมัครเรียนต่อและสมัครงาน นักเรียน — จบแต่ละชั้นปี ⭐ สำคัญ TCAS กสพท
2
ใบแสดงวุฒิการศึกษา
Diploma / Certificate
 ใบประกาศนียบัตร รับรองการจบ ม.3 (ภาคบังคับ) และ ม.6 (ขั้นพื้นฐาน) นักเรียน — จบ ม.3 / ม.6 เท่านั้น ⭐ สำคัญ กสพท
3
ใบรายงานผู้สำเร็จการศึกษา
Graduation Report
 รายชื่อผู้จบการศึกษา ส่งจากโรงเรียน → เขตพื้นที่ → กระทรวงศึกษา เพื่อรับรองวุฒิ โรงเรียน → สำนักงานเขต → กระทรวง เอกสารราชการ
4
รายงานผลการพัฒนาคุณลักษณะ
Character Development Report
 รายงานคุณธรรม จริยธรรม และพฤติกรรมของนักเรียน มักออกควบคู่กับ ปพ.1 นักเรียน — ควบคู่ ปพ.1 ประเมินพฤติกรรม
5
แบบบันทึกพัฒนาการนักเรียน
Student Development Record
 บันทึกพัฒนาการด้านการเรียน พฤติกรรม และการมีส่วนร่วมในกิจกรรม จากการสังเกตของครู ครูผู้สอน ครูดูแล
6
สมุดพก
Student Report Card
 รวบรวมผลการเรียนและพัฒนาการของนักเรียน จัดส่งให้ผู้ปกครองรายเทอมหรือรายปี นักเรียน / ผู้ปกครอง — รายเทอม/ปี รายเทอม
7
ใบรับรองผลการศึกษา
Temporary Enrollment Certificate
 เอกสารชั่วคราวรับรองความเป็นนักเรียน ใช้แทน ปพ.1 ในกรณีที่ยังออกไม่ได้ ⏰ อายุ 120 วัน นักเรียน — ชั่วคราว ก่อนได้ ปพ.1 ⭐ สำคัญ TCAS 120 วัน
8
ใบระเบียนสะสม
Cumulative Record
 รวบรวมข้อมูลทุกอย่างของนักเรียน ทั้งการเรียน สุขภาพ ผลงาน เพื่อใช้แนะแนว โรงเรียน — เก็บไว้ภายใน ฐานข้อมูลโรงเรียน
9
สมุดบันทึกผลการเรียนรู้
Learning Record Book
 รวบรวมวิชาและผลการเรียนแต่ละชั้นปี ใช้ในกรณีเทียบโอนหน่วยกิตเมื่อย้ายโรงเรียน นักเรียน — กรณีย้ายโรงเรียน เทียบโอน
⭐ ใบสำคัญ

3 ใบที่ต้องรู้ก่อน TCAS

```
1
📋
ใบ ปพ.1 — Transcript
ใบแสดงผลการเรียนอย่างเป็นทางการ ใช้สมัคร myTCAS และ กสพท ต้องขอจากโรงเรียนหลังจบแต่ละชั้นปี

myTCAS กสพท
2
🎓
ใบ ปพ.2 — ประกาศนียบัตร
ใบรับรองวุฒิ ได้รับเมื่อจบ ม.3 และ ม.6 เท่านั้น ใช้สมัคร กสพท และยืนยันวุฒิการศึกษา

กสพท จบ ม.3 / ม.6
7
🕐
ใบ ปพ.7 — ใบรับรองชั่วคราว
ใช้แทน ปพ.1 เมื่อยังออกไม่ได้ มีอายุ 120 วัน ใช้สมัคร myTCAS และ กสพท ได้เช่นกัน ต้องขอใหม่ถ้าหมดอายุ

myTCAS 120 วัน
```
🗺️ Mind Map

ลำดับการขอและใช้ใบ ปพ.

```
1

📚 ระหว่างเรียนทุกชั้นปี (รายเทอม / รายปี)

โรงเรียนจัดทำเอกสารติดตามพัฒนาการของนักเรียนอยู่ภายใน ไม่ต้องขอเอง

ปพ.5 แบบบันทึกพัฒนาการ (ครูดูแล) ปพ.6 สมุดพก (รับรายเทอม) ปพ.8 ระเบียนสะสม (โรงเรียนเก็บ)
2

🎓 จบการศึกษา ม.3 หรือ ม.6

ได้รับเอกสารสำคัญที่รับรองวุฒิการศึกษา

ปพ.1 ใบแสดงผลการเรียน (Transcript) ปพ.2 ใบประกาศนียบัตร (วุฒิ ม.3 / ม.6) ปพ.4 รายงานคุณลักษณะ (ควบคู่ ปพ.1) ปพ.3 โรงเรียนส่งกระทรวง (ตรวจสอบวุฒิ)
3

⏳ ช่วงรอ ปพ.1 — ยังไม่ได้รับ

ขอใบชั่วคราวจากโรงเรียน ใช้งานได้เหมือนกัน แต่มีอายุจำกัด

ปพ.7 ใบรับรองชั่วคราว (อายุ 120 วัน)
4

📝 ใช้สมัคร myTCAS / กสพท

ใช้ ปพ.1 หรือ ปพ.7 สมัคร myTCAS ส่วน กสพท ต้องใช้ทั้ง ปพ.1 / ปพ.7 และ ปพ.2 ด้วย

✅ myTCAS → ปพ.1 หรือ ปพ.7 ✅ กสพท → ปพ.1 + ปพ.2 (หรือ ปพ.7)
5

🔄 กรณีพิเศษ: ย้ายโรงเรียน

ต้องนำเอกสารบันทึกผลการเรียนทุกรายวิชาไปเทียบโอนหน่วยกิต

ปพ.9 สมุดบันทึกผลการเรียนรู้ (เทียบโอน)
```
